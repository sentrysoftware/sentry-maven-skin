## Overview

**${project.name}** is a modern [Apache Maven site](https://maven.apache.org/plugins/maven-site-plugin) skin for generating professional technical documentation websites from Markdown files.

### Key Features

- **Modern Design**: Clean HTML5/Bootstrap 3 responsive layout
- **Dark Mode**: Automatic detection with manual toggle
- **Full-Text Search**: Built-in search index (no external services)
- **Code Highlighting**: 290+ languages with copy-to-clipboard
- **Print-Ready**: Optimized CSS for printing
- **Mobile-First**: Responsive design for all devices
- **Image Optimization**: Automatic WebP conversion
- **AI-Ready**: Generates `llms.txt` for AI platforms
- **Fully Configurable**: Enable, disable, or customize every feature
